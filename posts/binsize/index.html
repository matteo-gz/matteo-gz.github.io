官方自带

go build -ldflags '-s -w'

upx 降低

https://github.com/upx/upx 要选择最新版本,有优化效果

upx -9 ./your-bin

减少包依赖大小

大小分析

https://github.com/Zxilly/go-size-analyzer

# download
go install github.com/Zxilly/go-size-analyzer/cmd/gsa@latest
# 分析大小
gsa --web ./your-bin
